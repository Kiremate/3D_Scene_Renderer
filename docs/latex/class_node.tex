\doxysubsection{Detailed Description}
Class representing a node in a scene graph. 

The \doxylink{class_node}{Node} class is designed to be part of a scene graph, containing transformation information as well as a list of child nodes. It\textquotesingle{}s capable of constructing a hierarchical representation of a 3D scene, where each node can have many children but only one parent.
